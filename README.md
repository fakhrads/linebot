## What Is This?

LINE BOT UnOfficial using linepy library.

This bot use 4 line accounts

## How to install ?

```sh
$ sudo apt-get update
$ sudo apt-get install python3 python3-pip git
$ git clone https://github.com/fakhrads/linebot
$ cd linebot
$ pip3 install -r requirements.txt
```

## How to run ?

```sh
$ screen
$ python3 a.py
```
